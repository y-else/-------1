# 多分类待办事项管理系统

## 项目简介

这是一个基于 Vue3 CDN 构建的多分类待办事项管理单页应用，采用纯 HTML 单文件方式编写，无需任何构建工具。系统支持按分类管理待办事项，包含超时提醒功能，数据本地持久化存储。

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 页面结构标记 |
| CSS3 | 样式设计，Flex 布局，响应式适配 |
| Vue3 | 通过 CDN 引入，Options API 实现数据绑定 |
| localStorage | 待办数据持久化，刷新不丢失 |

## 功能特性

- **新增待办**：支持输入内容、选择分类、设置截止时间
- **删除单条**：点击删除按钮移除指定待办
- **标记完成**：勾选复选框标记任务完成状态
- **批量清除**：一键清除所有已完成任务
- **分类筛选**：全部/工作/生活/学习 三个分类快速切换
- **超时提醒**：未完成且超期的任务自动标红提醒
- **数据持久化**：所有数据保存到 localStorage，刷新不丢失
- **响应式设计**：适配手机端和电脑端

## 项目架构说明

项目采用**纯单文件**架构，所有代码都在一个 HTML 文件中：

- **`<head>` 部分**：包含 CSS 样式（全局重置、布局、组件样式、响应式）
- **`<body>` 部分**：包含 Vue3 模板结构，使用 `v-for`、`v-if`、`v-model`、`:class`、`@click`、`@change` 等指令
- **`<script>` 部分**：Vue3 应用逻辑

### 核心逻辑说明

1. **数据结构**：待办项包含 `id`、`text`、`category`、`deadline`、`done` 五个字段
2. **分类筛选**：通过 `computed` 计算属性 `filteredTodos` 实时过滤列表
3. **超时判断**：`isOverdue` 方法对比截止时间与当前时间，超时且未完成的任务添加红色样式
4. **数据持久化**：每次数据变更调用 `saveTodos` 保存到 localStorage，`mounted` 生命周期中调用 `loadTodos` 恢复数据
