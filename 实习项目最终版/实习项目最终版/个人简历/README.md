# 个人简历展示页面

## 项目简介

这是一个基于 Vue3 CDN 构建的个人简历展示单页应用，采用纯 HTML 单文件方式编写，无需任何构建工具、脚手架或包管理器。页面设计简洁大气。

## 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 页面结构标记 |
| CSS3 | 样式设计，包含 Flex 布局、响应式适配、主题变量 |
| Vue3 | 通过 CDN 引入，使用 Options API 实现数据绑定 |
| localStorage | 数据持久化（主题偏好、留言内容） |

## 功能特性

- **响应式布局**：适配手机端和电脑端，移动端自动调整布局
- **深色/浅色主题切换**：点击右上角按钮切换，偏好保存到本地
- **技能动态展示**：技能列表带进度条动画，技能标签动态渲染
- **留言表单**：支持访客留言，非空校验，数据本地存储不丢失
- **项目经历展示**：清晰展示项目信息、技术栈标签

## 页面结构

```
├── 头部个人信息区
│   ├── 头像
│   ├── 姓名、职位
```

## 项目架构说明

项目采用**纯单文件**架构，所有代码都在一个 HTML 文件中：

- **`<head>` 部分**：包含 CSS 样式（主题变量、布局、组件样式、响应式）
- **`<body>` 部分**：包含 Vue3 模板结构，使用 `{{ }}` 插值、`v-for`、`v-if`、`:class`、`:style`、`@click` 等指令
- **`<script>` 部分**：Vue3 应用逻辑，包含 data、computed、methods、mounted 生命周期

### 核心逻辑说明

1. **主题切换**：通过修改 `document.body` 的 class 来切换 CSS 变量，同时保存到 localStorage
2. **技能进度条**：使用 `v-for` 遍历技能数组，`:style` 动态绑定进度条宽度
3. **留言功能**：表单校验 → 添加到列表 → localStorage 持久化 → 页面刷新时从 localStorage 恢复
